Drug Combos

2016 — present

dillonjohnbrown.github.io/drug-combos

I found this great chart showing how (relatively) safe combinations of drugs can be. I knew I could use my web development know-how to improve it, so I made it into a web app.

My web app is a pet project of mine and is constantly evolving. I do a rudimentary form of user testing on my friends whenever I make improvements.

I also put it's source code on GitHub for anyone to use in any way they want. This is for harm reduction and education, not profit.
