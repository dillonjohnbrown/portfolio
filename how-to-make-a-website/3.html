"Computer! Make me a website!"

Ok, so computers don't speak English like you and I do. They only understand code. Website code is called HTML.

*Whew*, break time. Here is a photo of Max to balance out all this learning.




HTML looks like this:

<p>Hello</p>

That turns into this:

Hello

"Amazing. When do we get to the exciting stuff like watching paint dry?"

It's so simple! The <p> tells the computer start a paragraph, and the </p> tells it this is the end of the paragraph.

HTML doesn't get much more complicated than that. There are a zillion different things, or elements, that can be on a webpage. Here is an image:

<img src="images/3/max-and-friends.jpeg">

When a computer runs that code it turns it into this photo of Max and a couple of his doggy friends.

Some elements, like images, don't need a start and an end part. They are just one thing. Let's break it down real quick.

This < means "hey computer, here comes some HTML code."

> means this is the end of this thing, or element.

img means this element is an image.

src= tells the computer where the image is at.

"images/3/max-and-friends.jpeg" is the path to the image file. It is in a folder called images, and inside that is a folder called 3, and inside that is a file called max-and-friends.jpeg

One last thing

This <!-- --> is a comment. Anything between <!-- and --> won't show up on your page.
