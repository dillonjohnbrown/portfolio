CSS + SVG

I like CSS. I just kind of get it. Below is an example of an animation using CSS and an SVG file.

A little bit of Less goes a long way.

This is the source code for the animation. Abusing CSS like this is an absolute dog as far as performance goes, but not a single line of JavaScript was used.

@keyframes pulse {
  from,
  50%,
  to {
    opacity: 1;
  }
  25%,
  75% {
    opacity: 0;
  }
}

.pulse-loop (@i) when (@i > 0) {
  path:nth-of-type(@{i}) {
    animation-delay: ~"@{i}0ms";
  }
  .pulse-loop(@i - 1);
}

.pulse-loop (261);

.animateSVG:checked ~ svg path {
  animation-name: pulse;
}
.svg-animation-1 path {
  animation-duration: 20s;
  animation-fill-mode: both;
  animation-iteration-count: infinite;
  animation-timing-function: ease-in-out;
}
